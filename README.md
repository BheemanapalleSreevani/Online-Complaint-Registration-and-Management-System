# 🛡️ Online Complaint Registration and Management System

An efficient web-based complaint management platform that enables users to register complaints online, track their status, and allows administrators to manage, assign, and resolve complaints through a centralized dashboard.

---

## 📖 Overview

The **Online Complaint Registration and Management System** is designed to digitize the complaint handling process by providing an easy-to-use interface for users and administrators. It eliminates manual complaint registration, improves transparency, and enables faster issue resolution.

---

## ▶️ Run the Application

### Start Backend

```bash
cd server
npm start
```

### Start Frontend

```bash
cd client
npm start
```

The application will run at:

- Frontend: `http://localhost:3000`
- Backend: `http://localhost:5000`

## 🔄 Workflow

1. Register or Login.
2. Submit a Complaint.
3. Complaint is stored in MongoDB.
4. Administrator reviews the complaint.
5. Complaint status is updated.
6. User tracks the complaint until resolution.
